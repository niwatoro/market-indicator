# 市場指標ビュワー

## 概要

デスクトップで常駐してマーケットの指標を表示します。対応している指標は以下の通りです。

- 株式指標
  - S&P500
  - NASDAQ
  - 日経平均
  - 上海総合
- 為替
  - ドル円
  - ユーロドル
- 金利
  - 米国10年債
  - 日本国債10年
  - ドイツ国債10年
  - 中国国債10年
- 暗号資産
  - ビットコイン
- ニュース
  - Bloomberg
  - 日経新聞

## 使い方

1. ライブラリをインストールする

```bash
npm install
```

2. アプリケーションを起動する

```bash
npm start
```

3. アプリをビルドする

```bash
npm run make
```

## 備考

MacOSでビルドしたアプリが隠しファイルにされることがある。その場合は以下のコマンドを実行して隠しフラグを外すこと。

```bash
chflags nohidden /path/to/your_app.app
```
